# RuoYi AI Quick Starter 网站

这是 RuoYi AI Quick Starter 项目的官方网站。

## 文件说明

- `index.html` - 网站主页
- `admin.png` - Web 管理界面预览图
- `mobile.png` - 移动端登录页面预览图
- `logo.svg` - 项目 Logo
- `tech/` - 技术栈图标目录

## 使用方式

直接在浏览器中打开 `index.html` 即可查看网站。

## 部署

可以将整个 `website` 目录部署到任何静态网站托管服务，如：
- GitHub Pages
- Netlify
- Vercel
- 传统 Web 服务器

## 技术栈

- HTML5
- Tailwind CSS (CDN)
- Font Awesome (CDN)
- 纯静态页面，无需构建
